# FinGrow
- This application assists users in managing their finances by tracking income, expenses, and savings goals, providing valuable insights into their spending habits and promoting better financial decisions.

## Features

1. Multi-User Support: The application can manage multiple users, each with their own financial data.
2. Transaction Tracking: Records and tracks individual transactions for each user.
3. Budget Management: Allows users to set and monitor budgets for different categories, showing how much has been spent in each.
4. Savings Goal Tracking: Enables users to define savings goals and track their progress over time.
5. Financial Reports: Generates detailed reports summarizing total income, expenses, budget usage by category, and savings goal progress over a defined period.
6. Alerts & Notifications: Notifies users when they exceed their budget in any category or are close to achieving a savings goal.

## Technologies Used
- Typescript
- Jest
## Requirements
1. Install Node.js on macOS.
   ```bash
   brew install node
   ```
2. Install watchman on macOS using brew.
   ```bash
   brew install watchman
   ```
## Installation
1. To clone the repository:
   ```bash
   git clone -b fingrow-part-1 https://github.com/vinaysaieverest/fin-grow
   ```
2. To install dependencies:
   - Go to the project directory
      ```bash
      cd folder-name
      ```
   - Run following command.
      ```bash
      npm install
      ```
